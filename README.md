# PC File Analyzer

## 프로젝트 소개

PC File Analyzer는 사용자가 입력한 폴더 경로를 기준으로 하위 폴더까지 탐색하여 파일과 폴더 정보를 분석하는 Python CLI 프로그램입니다.

이 프로젝트는 Python을 탑다운 방식으로 학습하기 위해 진행한 실습 프로젝트입니다. 단순히 문법을 먼저 학습하는 방식이 아니라, 실제로 필요한 기능을 구현하면서 `pathlib`, 반복문, 조건문, 딕셔너리, 리스트, 정렬, 파일 크기 계산 등을 학습하는 것을 목표로 합니다.

## 주요 기능

- 입력한 폴더 경로가 존재하는지 확인
- 입력한 경로가 폴더인지 확인
- 하위 폴더까지 포함하여 전체 파일 개수 계산
- 하위 폴더까지 포함하여 전체 폴더 개수 계산
- 확장자별 파일 개수 통계 출력
- 전체 파일 용량 합계 출력
- 용량이 큰 파일 TOP 10 출력

## 사용 기술

- Python 3.13
- pathlib
- Git / GitHub
- PyCharm
